Other platforms & older versions

Windows Installation

  1. Run the downloaded .exe installer
  2. If SmartScreen appears → Click "More info" → "Run anyway"
  3. Or: Right-click → Properties → Check "Unblock" → OK → Run
  4. App will install and create a desktop shortcut

macOS Installation

  1. Right-click the .dmg → Open (don't double-click)
  2. If blocked: System Settings → Privacy & Security → "Open Anyway"
  3. Or run: xattr -cr ~/Downloads/HackLayer-CTF-*.dmg
  4. Drag to Applications folder

Linux Installation

  1. Make executable: chmod +x HackLayer-CTF-*.AppImage
  2. Run: ./HackLayer-CTF-*.AppImage
